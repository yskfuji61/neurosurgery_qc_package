# 条件・閾値・時間条件ハブ 初版

## 結論
薬剤名や疾患名よりも、時間条件・検査値・処置予定が安全性を左右する。

| 条件 | 関連領域 | 誤読防止 |
|---|---|---|
| 受傷3時間以内 | TBI/TXA | 3時間超・時刻不明では機械的候補化しない |
| 動脈瘤閉鎖前 | aSAH/TXA/ε-ACA | 閉鎖後漫然継続しない |
| 原則24時間以内 | aSAH抗線溶薬 | 長期投与として扱わない |
| 最終服用時刻 | 抗凝固薬・抗血小板薬 | 薬剤名だけで中和・輸血候補化しない |
| 腎機能 | DOAC、TXA、LMWH、VCM、ACV、CFPM等 | 未入力なら薬剤部確認・注意表示 |
| PT-INR/aPTT/ACT | VKA/UFH/処置前 | 検査未確認で補正薬候補を出さない |
| 血小板数 | 血小板減少、処置前 | 抗血小板薬内服歴と混同しない |
| フィブリノゲン値 | rt-PA後、大量出血、術中出血 | 閾値は輸血部基準で確認する |
| 血清Na・浸透圧・尿量 | ICP管理、DDAVP、浸透圧療法 | 一般病棟標準運用にしない |
| GCS・CT所見 | TBI、重症度判定 | TBIなら一律TXAではない |

## 原資料からは確定できないこと
各閾値の最終値、採血頻度、投与中止基準、病棟別運用は原資料からは確定できない。
