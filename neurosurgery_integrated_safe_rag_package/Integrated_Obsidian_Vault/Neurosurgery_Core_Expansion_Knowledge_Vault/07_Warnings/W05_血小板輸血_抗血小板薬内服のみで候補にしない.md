# W05 血小板輸血：抗血小板薬内服のみで候補にしない

## 結論
血小板輸血は、抗血小板薬内服のみで一律候補化しない。

## 原資料要約
原資料では、抗血小板薬内服のみで血小板輸血を施設内手順候補にせず、緊急開頭・EVD等の侵襲的処置予定の有無を最初の分岐にすると整理されている。

## GPTによる整理・解釈
抗血小板薬曝露、血小板減少、緊急処置予定、輸血部基準は別条件である。血小板に関係するという理由だけで一括化しない。

## 標準化しない事項
- 抗血小板薬内服だけで血小板輸血を候補にしない。
- 血小板数や処置予定を確認せずに候補化しない。
- 輸血部基準や夜間休日体制を未確認のまま候補化しない。
