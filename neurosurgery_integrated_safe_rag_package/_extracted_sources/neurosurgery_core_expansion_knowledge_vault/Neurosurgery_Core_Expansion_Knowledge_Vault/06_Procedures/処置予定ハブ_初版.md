# 処置予定ハブ 初版

## 結論
処置予定ノートは、休薬・中和・補正・感染予防・VTE再開を横断するハブである。

| 処置予定 | 横断する論点 | 誤読防止 |
|---|---|---|
| 緊急開頭術 | 抗凝固薬中和、抗血小板薬、血小板数、凝固検査、予防抗菌薬 | 疾患名だけで薬剤を決めない |
| EVD | 挿入/抜去前後の抗血栓薬、凝固・血小板、感染予防、VTE再開 | EVD全留置期間の予防抗菌薬継続を標準化しない |
| EVT | rt-PA、抗血小板、抗凝固、画像確認、腎機能 | DAPT/bridgingを公開資料だけで候補化しない |
| クリッピング | aSAH閉鎖前後、抗線溶薬、抗菌薬、VTE | 閉鎖後TXAを漫然継続しない |
| コイル塞栓術 | 閉鎖前後、ステント有無、抗血小板薬 | 抗血小板薬を自動候補化しない |
| シャント関連処置 | 予防抗菌薬、感染治療、抗血栓管理 | 予防と治療を混同しない |
| 保存的管理 | 再評価、VTE予防、抗血栓薬再開 | 処置前補正と同じ扱いにしない |
| ICU/SCU/HCU管理 | 浸透圧療法、鎮静、重症出血、採血・神経観察 | 一般病棟と同じ運用にしない |

## 原資料からは確定できないこと
処置別の血小板数・凝固検査閾値、術者判断、輸血部基準、抗菌薬投与時間、抗凝固再開時期は施設内確認が必要である。
