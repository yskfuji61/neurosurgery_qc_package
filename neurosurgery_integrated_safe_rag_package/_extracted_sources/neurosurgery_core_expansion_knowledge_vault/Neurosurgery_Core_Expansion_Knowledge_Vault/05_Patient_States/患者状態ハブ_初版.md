# 患者状態ハブ 初版

## 結論
患者状態ノートは、薬剤選択前に参照される安全確認ハブである。

| 患者状態 | 必須確認事項 | 誤読防止 |
|---|---|---|
| 抗凝固薬曝露 | 薬剤名、最終服用時刻、腎機能、PT-INR/aPTT/ACT、手術/EVD予定 | ワルファリン、DOAC、UFH、LMWHを一括しない |
| 抗血小板薬曝露 | 薬剤名、DAPT、最終内服、血小板数、手術/EVD予定 | 内服のみで血小板輸血候補にしない |
| 血栓溶解薬曝露 | 投与時刻、画像、出血症状、フィブリノゲン | rt-PA後出血対応と二次予防を混同しない |
| 腎機能低下 | Cr/eGFR/CrCl、尿量、透析 | DOAC、TXA、LMWH、VCM、ACV、CFPM等で確認必須 |
| 肝機能低下 | AST/ALT、PT-INR、凝固異常 | 薬剤再開や抗てんかん薬選択を機械的に決めない |
| 血小板減少 | 血小板数、原因、処置予定 | 抗血小板薬曝露と混同しない |
| 凝固異常 | PT-INR、aPTT、Fib、薬剤歴、肝疾患 | 検査未確認で補正薬候補を出さない |
| 血栓症高リスク | AF、DVT/PE、機械弁、悪性腫瘍等 | 出血対応と抗凝固再開判断を混ぜない |
| けいれんリスク | 発作歴、病変、術式、腎肝機能、EEG要否 | 予防投与と治療投与を混同しない |
| 夜間休日 | 在庫、薬剤部、輸血部、看護観察体制 | 実行不能な手順を標準化しない |

## 原資料からは確定できないこと
患者状態ごとの最終アラート条件、検査閾値、病棟別運用、責任者承認は施設内確認が必要である。
