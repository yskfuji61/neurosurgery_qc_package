# 施設内確認チェックリスト 初版

| 項目 | 確認内容 |
|---|---|
| 施設採用品 | 対象薬剤・規格・代替薬 |
| 夜間休日在庫 | 高リスク薬、輸血製剤、抗菌薬、浸透圧療法薬 |
| 薬剤部調製 | 調製方法、払い出し、ダブルチェック、時間外対応 |
| 輸血部運用 | 血小板、FFP、クリオ、フィブリノゲンの基準と在庫 |
| 看護観察体制 | 神経所見、Na、浸透圧、尿量、投与中観察 |
| ICU/SCU/HCU/一般病棟運用 | 病棟別に実行可能な投与・採血・観察 |
| 委員会承認 | 薬剤部、医療安全、診療科、感染制御、輸血部、情報部門 |
| 保険請求方針 | 適応外・高額薬・緊急使用時の理由記録 |
| CDS実装可否 | 情報提示、注意喚起、入力制御、監査ログ、権限 |

## 禁止
