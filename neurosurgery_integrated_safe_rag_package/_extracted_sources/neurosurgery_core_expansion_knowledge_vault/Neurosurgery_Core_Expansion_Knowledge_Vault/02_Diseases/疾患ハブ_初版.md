# 疾患ハブ 初版

## 結論
疾患ノートは、薬剤推奨リストではなく、患者状態・処置予定・条件・閾値・陰性知識へ接続するハブとして扱う。

## 中核疾患

| 疾患・病態 | 主な接続先 | 誤読防止の要点 |
|---|---|---|
| 自然発症脳内出血 | 抗凝固薬曝露、抗血小板薬曝露、緊急開頭、EVD、W01〜W05 | TXA、rFVIIa、DDAVP、血小板輸血を条件確認なしに候補化しない |
| 外傷性脳損傷 | 受傷時刻、GCS、CT所見、TXA、ICP管理 | TBIならTXAと短絡しない。受傷3時間以内など条件を必須化する |
| 動脈瘤性くも膜下出血 | 動脈瘤閉鎖前後、TXA/ε-ACA、クリッピング、コイル | 抗線溶薬は閉鎖前短期橋渡し候補。閉鎖後漫然継続しない |
| 慢性硬膜下血腫 | 再発予防、抗血栓薬曝露、転倒リスク | CSDH再発予防の情報を急性ICH/TBI止血へ横展開しない |
| 急性虚血性脳卒中 | rt-PA、EVT、24時間後画像確認、抗血栓薬再開 | 急性期治療、出血時対応、二次予防を混同しない |
| 脳浮腫・ICP上昇 | 高張食塩液、マンニトール、Na、浸透圧、ICU/SCU | 監視条件なしに一般病棟標準運用しない |
| 中枢神経感染症・周術期感染 | 予防抗菌薬、髄膜炎、EVD/シャント感染、脳炎疑い | 予防投与と感染治療を混同しない |
| VTE予防・抗凝固再開 | UFH/LMWH/DOAC/ワルファリン、画像安定、処置後 | 中和と再開を同一フローにしない |

## 原資料からは確定できないこと
各疾患における施設内標準手順、投与量、閾値、運用可否、電子カルテ表示仕様は原資料だけでは確定できない。
