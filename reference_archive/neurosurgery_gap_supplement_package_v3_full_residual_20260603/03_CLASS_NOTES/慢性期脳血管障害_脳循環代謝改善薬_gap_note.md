# 慢性期脳血管障害・脳循環代謝改善薬 gap note

## 結論

この領域は、添付文書上の効能がある薬剤、過去に使われた薬剤、再評価で有用性が問題になった薬剤、販売・承認状況が不明な薬剤が混在する。RAGは、これらを同列の「推奨薬」として提示してはならない。

## Safe RAGルール

「脳梗塞後遺症に伴うめまい・意欲低下」は、急性期再灌流療法、抗血栓療法、脳保護薬、SAH術後血管攣縮薬とは別レイヤーで扱う。
