# V3統合手順

1. 元ZIPとV2パッケージをバックアップする。
2. `09_MANIFESTS/neurosurgery_gap_drug_supplement_master_V3_ALL.csv` を既存master候補と突合する。
3. `drug_key` の重複を確認し、既存profileがある薬剤は文脈noteとして統合する。
4. `pmda_product_source_register_v3_residual.csv` の UNRESOLVED_DO_NOT_GUESS をPMDA製品単位で解決する。
5. PMDA添付文書、IF、RMP、患者向医薬品ガイド、審査/再審査報告書、販売中止・経過措置・薬価削除を確認する。
6. 院内採用品、調製可否、プロトコル、専門医承認が必要な薬剤をフラグ化する。
7. `08_VALIDATION_CHECKS/v3_residual_rag_safety_tests.csv` をRAGに流し、誤回答を記録する。

## 統合時の禁止事項

- PMDA URLを推測で埋めない。
- procedural noteを標準治療薬profileとして扱わない。
- historical / negative noteを現役推奨薬として扱わない。
- 国内薬事、保険、院内運用、ガイドライン推奨を混同しない。
