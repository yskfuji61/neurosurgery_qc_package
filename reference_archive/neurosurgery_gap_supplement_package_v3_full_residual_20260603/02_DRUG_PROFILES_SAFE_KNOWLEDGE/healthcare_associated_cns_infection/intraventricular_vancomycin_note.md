# バンコマイシン脳室内投与note（intraventricular vancomycin note）

## Safe RAGでの位置づけ

脳室炎・シャント感染の特殊投与

この項目を追加した理由：静注バンコマイシンと混同不可。施設手順必須。

## 商品名例

バンコマイシン

## 安全上の焦点

感染症科/脳外/薬剤部承認、濃度、無菌調製、神経毒性

## RAGで禁止する誤回答

- PMDA製品単位の添付文書、RMP、IF、販売状況、院内採用品を確認せずに、標準治療として断定しない。
- 疾患名だけを根拠に投与可否を一般化しない。
- 国内薬事、保険、ガイドライン推奨、院内プロトコル、実際の処方可否を混同しない。
- procedural / specialist note の薬剤は、専門医判断・施設手順必須として回答する。

## 必須確認項目

1. PMDA添付文書の効能効果、用法用量、禁忌、重要な基本的注意。
2. RMP、患者向医薬品ガイド、インタビューフォームの有無。
3. 腎機能、肝機能、妊娠授乳、小児高齢者、相互作用、周術期休薬。
4. 院内採用品、調製手順、投与経路、監視項目、緊急時対応。
5. ガイドライン推奨の強さと、添付文書上の効能効果の差分。

## ステータス

PMDA製品単位未解決。手作業監査が必要。
